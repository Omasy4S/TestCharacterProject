## 📁 Структура проекта

```
ConsoleApp1/
├── Models/               # Модели персонажей
│   ├── Character.cs     # Абстрактный базовый класс
│   ├── Warrior.cs       # Класс воина
│   └── Mage.cs          # Класс мага
│
├── Weapons/              # Система оружия
│   ├── IWeapon.cs       # Интерфейс оружия
│   ├── Sword.cs         # Меч (физический урон)
│   ├── MagicStaff.cs    # Магический посох
│   └── FoolStaff.cs     # Жезл дурака (хаотичное оружие)
│
├── Core/                 # Игровой движок
│   ├── GameEngine.cs    # Главный игровой движок
│   └── CombatSystem.cs  # Система боя
│
└── Program.cs            # Точка входа
```

## 🚀 Как запустить

### **Требования:**
- .NET Framework 4.x или .NET 6+
- Visual Studio / Rider / VS Code

### **Запуск:**
1. Откройте проект в IDE
2. Постройте проект (`Build` → `Build Solution`)
3. Запустите (`F5` или `Ctrl+F5`)
